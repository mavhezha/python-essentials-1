# Python Essentials 1

Exercises and practice scripts from the [Cisco Networking Academy – Python Essentials 1](https://skillsforall.com/course/python-essentials-1) course, used as a refresher before diving into automation scripting.

---

## Files

| File | Description |
|------|-------------|
| `first.py` | Hello World and basic print/input operations |
| `py_literal_section.py` | Python literals – integers, floats, strings, booleans |
| `conditions_loops.py` | `if/elif/else` conditions and `for`/`while` loops |
| `k_word_argument.py` | Keyword and positional function arguments |
| `fib.py` | Fibonacci sequence implementation |
| `secret_number.py` | Number guessing game using conditionals and loops |
| `tic-tac-toe-raw.py` | Tic-tac-toe game using basic Python only |
| `tic-tac-toe-with-randomPackage.py` | Tic-tac-toe with computer moves using the `random` module |

---

## Getting Started

No dependencies required — just Python 3.

```bash
git clone https://github.com/mavhezha/python-essentials-1.git
cd python-essentials-1
python fib.py
```
